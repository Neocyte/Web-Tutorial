Cascading Style Sheet

CSS stands for cascading style sheet, which is used to style the content of the page.

Link a ".css" file to your HTML using
<link href="app.css" rel="stylesheet">

See the Pen CSS Syntax by Mason Wu (@neocyte) on CodePen.

Syntax

The basic syntax for CSS statements involves selectors and properties.

Selectors select what we want to style:
div {properties}

There are many types of selectors:

● HTML Selectors - all elements with this HTML tag

● Class Selectors - all elements with this class attribute

● ID Selectors - this unique element

● Context Selectors - this element that is the child of the parent element

● Grouped Selectors - multiple elements

Properties are the styles we want to apply:
div {color: black;}

See the Pen Pseudo Classes by Mason Wu (@neocyte) on CodePen.

Pseudo Classes

Pseudo classes allow you to apply styles during certain events

:hover
Used to style the content of an element when the mouse is hovering over it

:active
Used to select and style the active element. An element is active when you click or hold it

:link
Used to style unvisited links

:visited
Used to style visited links

:first-child
Selects this element that is the first child of its parent
