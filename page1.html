Hyper Text Markup Language

HTML stands for hyper text markup language, which is used to structure the content of the page.

Every page must start with a declaration statement <!DOCTYPE html> so that the browser can recognize its HTML.

See the Pen Syntax by Mason Wu (@neocyte) on CodePen.

Syntax

The basic syntax for HTML elements involves tag pairs.

The starting one is a command in angle brackets. The end has the same command with a slash in front of it. The content is between the two tags.

A tag may also contain attributes that further modify the element.

See the Pen Head And Body by Mason Wu (@neocyte) on CodePen.

Two Important Sections

An HTML document has two important sections:

● Head Section - provides important information to the search engine and browser.

● Body Section - contains all of the content

See the Pen Tags by Mason Wu (@neocyte) on CodePen.

Tags

Paragraph <p></p>
Displays text; changes line before and after content and gives empty line before and after

Div <div></div>
Wraps entire sections; changes line before and after content,

Span <span></span>
Displays text; wraps small pieces of text

Break Line <br>
Forces a new line

Heading <h1></h1>
Displays heading text

Link <a href="URL">Link</a>
Displays a link to a target URL

Image <img src="URL" alt="failed_image"/>
Displays an image using its source path. If the image fails to load, the text of the ALT attribute will be shown instead
